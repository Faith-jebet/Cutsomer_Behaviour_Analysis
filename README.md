# Cutsomer_Behaviour_Analysis
Data analytics project showcasing customer behavior analysis using python, sql and power BI

## Overview

This project demonstrates a complete data analytics workflow using Python, PostgreSQL, SQL, and Power BI. The objective is to analyze customer behavior by transforming raw data into meaningful business insights through data cleaning, exploratory data analysis (EDA), SQL analysis, and interactive visualizations.

The project follows an end-to-end analytics process commonly used in real-world business environments.

---

## Dataset

The dataset contains customer-related information, including demographics, purchasing behavior, and transaction details. It is used to identify customer trends, spending patterns, and business performance metrics.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Jupyter Notebook.
* Loaded the data using the Pandas library.

### 2. Exploratory Data Analysis (EDA)

* Explored dataset structure.
* Generated summary statistics.
* Identified missing values.
* Examined distributions and relationships between variables.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Converted data into appropriate formats.
* Prepared the dataset for analysis.

### 4. PostgreSQL & SQL Analysis

* Connected Python to PostgreSQL.
* Imported the cleaned dataset into the PostgreSQL database.
* Executed SQL queries to answer business questions.
* Performed filtering, grouping, aggregation, sorting, and other analytical queries.

### 5. Power BI Dashboard

Created an interactive dashboard to visualize key business metrics, including:

* Customer demographics
* Sales performance
* Customer purchasing behavior
* Revenue trends
* Key Performance Indicators (KPIs)

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/customer-behavior-analysis.git
cd customer-behavior-analysis
```

### 2. Install the required Python packages

```bash
pip install -r requirements.txt
```

### 3. Open the Jupyter Notebook

```bash
jupyter notebook
```

Run the notebook to perform data loading, cleaning, and exploratory data analysis.

### 4. Set up PostgreSQL

* Create a PostgreSQL database.
* Import the cleaned dataset.
* Execute the SQL queries located in the `sql/` folder.

### 5. Open the Power BI Dashboard

Open the `.pbix` file using Microsoft Power BI Desktop to explore the interactive dashboard.
